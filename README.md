# 🚦 Smart Traffic Control System

## 📌 Overview
This project simulates a smart traffic signal system that dynamically adjusts signal timing based on traffic density and waiting time.

## ⚙️ Features
- Dynamic signal timing
- 4-way intersection simulation
- Vehicle movement visualization
- Smart signal selection

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Spring Boot

## ▶️ How to Run

### Backend
cd backend  
mvn spring-boot:run  

### Frontend
cd frontend  
npm install  
npm start
